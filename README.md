# Judee OJ前端项目
> Vue CLI 3 多页应用项目, 网站配色为[Nord](https://www.nordtheme.com/)，ElementUI为UI组件库, D2Admin为后台管理模板


## 初始化
```bash
npm install
```

## 使用
* 开发：
    * 本地运行：`npm run serve`  
    * 修复代码格式错误：`npm run lint`
* 打包：
    * 开发环境: `vue-cli-service build`
